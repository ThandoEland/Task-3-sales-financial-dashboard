# Sales Financial Dashboard

## Project Overview
This project presents an interactive Sales Financial Dashboard created using Tableau. The dashboard helps business stakeholders analyze sales performance, profitability, expenses, and sales trends through interactive visualizations.

## Objectives
- Analyze sales performance using key performance indicators (KPIs).
- Visualize sales trends over time.
- Compare sales across different product lines.
- Create an interactive dashboard using filters.
- Support business decision-making through data visualization.

## Dataset
The dataset contains sales and financial information, including:
- Date
- Sales
- Profit
- Total Expenses
- Product Line
- Product Type
- Market
- State
- Margin
- Target Sales
- Target Profit

## Dashboard Features
- KPI Cards:
  - Total Sales
  - Total Profit
  - Total Expenses
- Sales Trend (Line Chart)
- Sales by Product Line (Bar Chart)
- Interactive Filters:
  - Date
  - Market
  - Product Line
  - Product Type
  - State

## Key Insights
- Sales performance can be monitored over time.
- Product lines can be compared to identify top-performing products.
- Profit and expenses can be tracked using KPI cards.
